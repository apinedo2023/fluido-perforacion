# App de Fluido de Perforación (Métrico)

## Funcionalidades
- Registro de propiedades del lodo.
- Consumos con concentración (kg/m³) y costo.
- Volúmenes y transferencias entre pozos.
- Preparaciones con concentración objetivo.
- Inventario con alertas por stock mínimo.
- Costos diarios y acumulados.
- Dashboard comparativo por pozo, zona y equipo.
- Exportaciones Excel/CSV y reporte PDF diario.

## Ejecutar
```
pip install -r requirements.txt
streamlit run app.py
```
